СКРИПТОЛОГ — БЫСТРЫЙ СТАРТ (Windows/Mac)
1) Убедитесь, что установлен Python 3.9+ (python --version).
2) Windows: дважды кликните run.bat (сам установит зависимости и запустит бота).
   Mac/Linux: в терминале из папки проекта: ./run.sh
3) Откройте вашего Telegram-бота и введите /start.

Файлы:
- skriptolog_bot.py — код бота
- requirements.txt — зависимости
- .env — настройки (BOT_TOKEN, CSV_PATH)
- run.bat / run.sh — запуск в один клик

Примечание: .env содержит ваш приватный токен — никому не пересылайте.
